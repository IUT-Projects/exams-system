# Exams-System

> Exams System - oop project for second semester.


## **Requirements**

* Any C++ compiler (gcc, clang)
* `Make` - if you want to use Makefile (optional)

## **Running**

Run by main.cpp

```shell
g++ main.cpp -o program && ./program # compile
./program # run executable file
```

Run my make:

```shell
make run
```
